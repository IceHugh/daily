# Ethereum Pectra 升级进入测试阶段 🚀

以太坊的下一个重大升级 **Pectra**（Prague-Electra）已经正式进入测试网阶段，这是一个重要的里程碑！

## 关键改进

Pectra 升级将带来多项重要改进：

1. **账户抽象改进** - 使用户钱包体验更加流畅
2. **验证者提款限制增加** - 从每天 4 个增加到 16 个
3. **EIP-7702** - 智能合约钱包标准

## 测试网时间表

- **Q1 2025** - Goerli 测试网开始
- **Q2 2025** - Sepolia 测试网
- **Q3 2025** - 主网上线

## 影响

这次升级将显著改善用户体验，尤其是在 DeFi 和 NFT 交易方面，交易成本预计将降低 20-30%。
